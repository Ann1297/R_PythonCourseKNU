# Lab 1

## Task 1

Створити змінні базових (atomic) типів. Базові типи: character, numeric, integer, complex, logical.

+ character
```R
ch <- 'a'
print(class(ch))
```
```R
[1] "character"
```

+ numeric
```R
n <- 78.9
print(class(n))
```
```R
[1] "numeric"
```

+ integer
```R
i <- 789L
print(class(i))
```
```R
[1] "integer"
```

+ complex
```R
c <- 78+9i
print(class(c))
```
```R
[1] "complex"
```

+ logical
```R
l <- TRUE
print(class(l))
```
```R
[1] "logical"
```
